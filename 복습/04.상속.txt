 Object 생성자
  Object.create -> 객체 생성 메서드
    (proto(상위 객체 prototype), 새로 정의할 객체 속성 정의);

    속성 : {
      value : 속성의 값 (기본자료형 + 객체 ),
      writable : true | false, - 객체 속성의 쓰기 권한을 통제, 
      enumerable : true | false, 
      set 함수() { -> setter },
      get 함수 (....) { -> getter },
    }

Object.freeze() - 객체 동결 : 속성의 삭제, 수정, 변경, 불가

Object.seal() -> 객체 밀봉 -> 속성의 삭제, 추가 X, 수정은 가능
